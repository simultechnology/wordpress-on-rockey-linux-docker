# wordpress on Rocky Linux

## middlewares

- apache
- php
- php-fpm

## start

```
docker-compose up -d
```

## Stop & delete

```
docker-compose down
```

## Go into amazon linux container

```
docker exec -it wordpress-on-rocky bash
```
